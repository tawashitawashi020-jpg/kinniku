# きんにくRPG Phaser版

このリポジトリには、JavaScript のゲームエンジン **Phaser 3** を用いて作成した高品質なトップダウンアクション RPG のプロトタイプが含まれています。

## ゲーム概要

- プレイヤー（緑色の四角）を矢印キーで移動させます。広いマップを自由に歩き回ることができます。
- `SPACE` キーで近接攻撃（パンチ）を行います。攻撃はプレイヤーの進行方向に発生します。
- 赤い敵をすべて倒すと勝利メッセージが表示されます。敵にぶつかるとプレイヤーの HP が減少し、0 になるとゲームオーバーです。
- 敵はプレイヤーを追いかける簡易 AI を備えています。

## ファイル構成

- `index.html` … ゲーム画面を表示する HTML。Phaser を CDN から読み込んでいます。
- `src/game.js` … ゲームロジックを実装した ES モジュール。
- `README.md` … このファイル。

## 実行方法

1. このフォルダを任意の Web サーバーでホストするか、`index.html` をブラウザで直接開きます。<br>
   - 直接ファイルを開く場合、攻撃時に音や外部アセットを読み込まないため問題なく動作します。
   - ローカルサーバーを使用する場合は次のコマンド例を参考にしてください（Pythonがインストールされている場合）。
     ```bash
     # プロジェクトのディレクトリに移動
     cd kinniku_phaser_game
     # Python 3 の http.server モジュールで起動
     python3 -m http.server -b 127.0.0.1 8000
     ```
2. ブラウザで `http://127.0.0.1:8000` を開きます。
3. ゲームが開始したら、矢印キーや `SPACE` キーでプレイしてみましょう。

## カスタマイズ案

- プレイヤーや敵のテクスチャを独自の画像に差し替える場合、`preload()` で `this.load.image()` を使って読み込み、`create()` でそれを使用してください。
- マップの広さや敵の数は `worldWidth`、`worldHeight`、`spawnEnemies()` の引数で調整できます。
- 攻撃方法や敵の行動アルゴリズムを変更して、さらに面白いバトルを実装することも可能です。
