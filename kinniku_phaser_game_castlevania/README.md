# きんにく横スクロールRPG Phaser版

このリポジトリには、JavaScript のゲームエンジン **Phaser 3** を用いて作成した横スクロールアクション RPG のプロトタイプが含まれています。筋肉キャラを操作してステージを右へ進み、出現する敵を倒しながら複数のステージをクリアしていきます。イメージとしては「悪魔城ドラキュラ」のような二足歩行のアクションゲームを目指しています。

## ゲーム概要

- プレイヤー（緑色の長方形）を矢印キーで操作します。`←` / `→` で左右に移動し、`↑` でジャンプします。
- `SPACE` キーで前方に近接攻撃を繰り出します。攻撃にはクールダウンがあるため連打はできません。
- 赤い敵は左右にパトロールしており、接触するとプレイヤーの HP が減ります。HP が 0 になるとゲームオーバーです。
- ステージの右端まで到達すると次のレベルへ進みます。全てのレベルをクリアすると勝利メッセージが表示されます。
- 各ステージには簡単な足場やジャンプ台が設置されています。敵を倒した後もステージ右端へ進まないと次へ進めません。

## ファイル構成

- `index.html` … ゲーム画面を表示する HTML。Phaser を CDN から読み込みます。
- `src/game.js` … ゲームロジックを実装した ES モジュール。横スクロールやジャンプ、攻撃、レベル遷移の処理が含まれています。
- `README.md` … このファイルです。

## 実行方法

1. このフォルダを Web サーバーでホストするか、`index.html` をブラウザで直接開きます。<br>
   - ローカルサーバーを使用する場合は次のコマンドを参考にしてください（Python がインストールされている場合）。
     ```bash
     # プロジェクトのディレクトリに移動
     cd kinniku_phaser_game
     # Python 3 の http.server モジュールで起動
     python3 -m http.server -b 127.0.0.1 8000
     ```
   - ブラウザで `http://127.0.0.1:8000` を開くとゲームが開始します。
2. ゲームが開始したら矢印キーと `SPACE` キーでプレイしてみてください。プレイヤーの HP は画面左上に表示されます。

## 追加アイデア

- レベル定義 (`levels` 配列) を編集することで、地面や足場の配置、敵の出現位置や巡回範囲を自由に変更できます。新しいオブジェクトを追加するだけでステージを増やせます。
- プレイヤーや敵のテクスチャは `preload()` で生成しているため、画像ファイルに置き換えることも容易です。`this.load.image()` で読み込み、`create()` で使用してください。
- 攻撃方法や敵の AI を改良し、武器や特殊攻撃、ボスキャラクターを追加することで、より本格的なアクションゲームに発展できます。
